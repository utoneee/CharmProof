# CharmProof — 壓克力製作模擬

單檔 HTML 的壓克力吊飾打樣預覽工具。自用預覽功能，不保證準確度。

**Charm**＝吊飾、**Proof**＝印刷打樣（校樣）——模擬透明壓克力吊飾印刷後的實際觀感，下單前先看一眼。

## 使用方式

開啟 `index.html` 即可，無需安裝、無外部相依。

1. 批次上傳圖檔（正面／白墨／反面／刀模，同尺寸已對位的透明 PNG，1:1 疊合；檔名含「白墨/反面/刀模/正面」會自動對號，其餘照選取順序補位）
2. 選板材與加工，拖曳可轉動檢視，雙擊回正

## 功能

- **三畫布渲染**：透射墨（CSS multiply 模擬染色玻璃）＋遮光板材＋刀模投影分層合成，正反面光學不對稱（直視變透明、隔板看變暗）
- **板材**：透明／自訂色（透・半透・不透三段）／霧白／鐳射／螢光四色（黃綠・粉・橘・藍）
- **白墨模式**：無／底層／夾層(雙面)——白墨形狀取自白墨圖，未上傳時退用設計圖輪廓
- **加工**：反光帶、閃粉（少/多）、碎光（小/大）
- **檢視**：六種背景（透格/桌墊/染彩/淺色/深色/自訂圖）、厚度側壁、底部影子、翻面（鏡像）、檢視大小調整

## 圖檔說明

- 白墨圖不透明處＝印白處；刀模實心區＝壓克力外形，不選則用橢圓
- 白墨「無」＝圖案半透明透背景；「底層」＝正面實色、背面見白；「夾層」＝雙面各印一圖（背面用「背面」圖檔）
- 壓克力顏色：板色（透明/自訂）與特殊材質互斥，點板色即取消材質
